Sample Models

for

 

Halpin, T., Evans K., Hallock, P. & MacLean, W., Database Modeling with Microsoft® Visio for Enterprise Architects (Morgan Kaufmann, 2003)

 

Visio document files for the sample models displayed in the Further Resources section of the above book are accessible below. Download these files to inspect other aspects of the models, such as notes, data types, and mapping settings for name generation, and to practice forward engineering from conceptual to logical to physical levels.

To download one of these files, proceed as follows:

1. Click the link on the filename to invoke the File Download dialog.
2. Click the Save button to download the file to your local PC.
    (Do NOT click the Open button)
3. In the SaveAs dialog, choose the folder in which you wish to save the file.
4. Click the Save button to save the file.

 

Note: In order to edit or work interactively with these models, you must have installed a copy of Microsoft Visio for Enterprise Architects.

 

Patient_CS.vsd        Conceptual schema for a medical patient application

Patient_LS.vsd         Logical schema mapped from the Patient_CS.vsd

BPtest2_CS.vsd       Alternative conceptual schema for blood pressure tests

BPtest2_LS.vsd        Logical schema mapped from BPtest2_CS.vsd

BPtest3_CS.vsd       Another alternative schema for blood pressure tests

BPtest3_LS.vsd        Logical schema mapped from BPtest3_CS.vsd

Sales_CS.vsd           Conceptual schema for sales invoice application

Sales_LS.vsd           Logical schema mapped from Sales_CS.vsd
